# 🎯 PFR Precision Grader & Fraud Detective

This tool automates the screening of applicants and detects suspicious patterns (SIM farms, bots, and speed-demons).

### 🚀 How to use:
1. **Upload Call List:** Drag in your CSV or XLSX export.
2. **Upload Screener:** Use the PFR Standard Template (Q | A | Qualify | Screen-out).
3. **Verify Rejections:** The app automatically pulls "Disqualify" logic.
4. **Consistency Checks:** Link 'Profile Gender' to 'Screener Gender' to catch liars.
5. **Run Audit:** Review the Red Metrics and the Phone Cluster Gallery.

### 🛡️ Fraud Flags:
- **Phone Cluster:** 10-digit match on mobile numbers.
- **Inconsistency:** Mismatch between Profile and Screener answers.
- **Speed:** Completed faster than 60 seconds
